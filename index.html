OS.js

JavaScript Web/Cloud Desktop Platform

View on Github Install now

A desktop implementation for your browser with a fully-fletched window manager, Application APIs, GUI toolkits and filesystem abstraction.

Please note that OS.js is under development and features are not complete

Open-source

OS.js is completely free and open-source which means you can contribute to the development or use the code as you like. View License

Fast, flexible and customizable

Features a simple and very flexible API so you can customize and add features easiliy.

Comes with a Virtual Filesystem to easily create/modify files across different storage services and providers. Includes support for storage on OS.js deployed servers and Google Drive.

OS.js is also very small and lightning fast.

Desktop

The Desktop and Window Manager is easy to use and fully customizable. You can change themes, wallpapers and modify panels.

Built to feel like a real desktop, only in the browser.

Applications and packages

Comes complete with a basic productivity suite: File Manager, Drawing Application, Music/Video Player, Calculator, Text-editor and a Richtext-editor.

You can easily add more packages using external repositories (to which you can contribute).

Compability

Works in any modern browser that supports HTML5 and ES5.1: Chrome, Firefox, Opera, Safari, Internet Explorer 10+

Compatible with Mobile devices using Android or iOS (But has currently no custom UI).

You can deploy a server on Linux, BSD, OSX and Windows/Cygwin using Nodejs or PHP (Apache and Lighttpd)

Installation

Just run the automated installer and you will be ready to go:

curl -sS http://andersevenrud.github.io/OS.js-v2/installer | sh

Make sure to read the installation documentation before you begin. There you will also find instructions on how to download and install manually.

Full list of features

  • Free and Open-source
  • Very simple installation
  • Works in any modern browser
  • Deployable on any platform using either PHP or Node
  • Dependency-free frontend
  • Easy to use API
  • Customizable and easy to extend with custom code and modules etc.
  • Virtual File System - Store your files across many different locations
  • Desktop and Window Manager built to feel familiar to most users
  • Drag-and-drop between applications
  • Supports sessions so you can reload workspaces on any computer
  • Localization and translations
  • Comes with a small application suite
  • Supports adding of packages via external repositories
  • Can be deployed using X11 to work as a full-fletched desktop solution
  • Comes with all the tools nesessry to build your own applications
  • Source-code up to strict standards and fully linted
