OS.js

JavaScript Web/Cloud Desktop Platform

Fork on Github Download latest source

Please note that OS.js is under development and features are not complete

Open-source

OS.js is completely free and open-source which means you can contribute to the development or use the code as you like. View License

Fast, flexible and customizable

Features a simple and very flexible API so you can customize and add features easiliy.

Comes with a Virtual Filesystem to easily create/modify files across different storage services and providers. Includes support for storage on OS.js deployed servers and Google Drive.

OS.js is also very small and lightning fast.

Desktop

The Desktop and Window Manager is easy to use and fully customizable. You can change themes, wallpapers and modify panels.

Applications and packages

Comes complete with a basic productivity suite: File Manager, Drawing Application, Music/Video Player, Calculator, Text-editor and a Richtext-editor.

You can easily add more packages using external repositories (to which you can contribute).

Compability

Works in any modern browser that supports HTML5 and ES5.1: Chrome, Firefox, Opera, Safari, Internet Explorer 10+

Compatible with Mobile devices using Android or iOS (But has currently no custom UI).

You can deploy a server on Linux, BSD, OSX and Windows/Cygwin using Nodejs or PHP (Apache and Lighttpd)
