File Tree Overview

This page describes the file-tree. It should be pretty self-explainatory with the naming and structure.

  • dist-dev

    This is the directory where the server runs from in development mode.

  • dist

    This is the directory where the server runs from in production mode.

  • doc

    Misc documentation files

  • src

    This is the source file directory

    • conf

      Configuration files used for building and settings used in OS.js

    • javascript

      JavaScript (frontend) sources

    • packages

      Packages (Application) sources

    • server-node

      Node.js server

    • server-php

      PHP server

    • stylesheets

      CSS Stylesheet (frontend) sources

    • themes

      Theme sources (icons, sounds, stylesheets)

    • tools

      Templates and libraries used for building

    • mime.json

      MIME (filetype) mapping table

  • vendor

    This is the vendor source file directory (third party etc)

  • vfs

    Directory for storing VFS files (user files)

  • Gruntfile.js

    Gruntfile. Used for building
