Adding Google API and VFS support

Simply update the configurations with the name, description and a path.

110-client.json

{
  "client" : {
    "VFS": {
      "Mountpoints": {
        "example": {
          "description": "Example"
        }
      }
    }
  }
}

120-server.json

{
  "server" : {
    "vfs": {
      "mounts": {
        "example": "/opt/example/directory"
      }
    }
  }
}

Then run: 

grunt config
