Adding Google API and VFS support

Simply create a config file to enable both API and VFS. Ex src/conf/901-gapi.json: 

{
  "settings" : {
    "frontend" : {
      "GoogleAPI": {
        "ClientId": "INSERT YOUR CLIENT ID HERE"
      },

      "Core" : {
        "VFS": {
          "GoogleDrive": {
            "Enabled": true
          }
        }
      }
    }
  }
}

Then run: 

make config
