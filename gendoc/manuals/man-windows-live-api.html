Adding Windows Live API and OneDrive VFS support

Simply create a config file to enable both API and VFS. Ex src/conf/901-gapi.json: 

{
  "settings" : {
    "frontend" : {
      "WindowsLiveAPI": {
        "ClientId": "INSERT YOUR CLIENT ID HERE"
      },
      "Core" : {
        "VFS": {
          "OneDrive": {
            "Enabled": true
          }
        }
      }
    }
  }
}

Just add your ClientID to build.json. If you want to enable the VFS module, just toggle the "enabled" option.

Then run: 

make config
