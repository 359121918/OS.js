Tutorial: Create your own Panel Item

You basically just have to copy one of the existing items (like Clock) and use it as a template. It is just one CSS file and one JS file.

Or if you want to create your own from scratch, extend OSjs.Applications.CoreWM.PanelItem and add it to the namespace OSjs.Applications.CoreWM.PanelItems

Include your files in either CoreWM manifest or in build.json
