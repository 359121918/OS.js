Tutorial: Create your own Handler

The handler is the basis for communication, resource handling, settings and sessions.

This tutorial only shows you how to use the included example handler.

This handler does basic login and server sessions. It will display a login window when you boot OS.js

You can find full documentation of the Handler methods in the Client API reference.

Add login HTML template

Copy these files: 

cp src/javascript/handlers/example/index.html dist/example.html
cp src/javascript/handlers/example/index.html dist-dev/example.html

Changing handler

You just have to edit src/conf/000-base.json and swap out the configured handler name.

Then run: 

grunt core config dist-dev-index

Setting up database

Create a new mysql user and database, then insert this table: 

CREATE TABLE IF NOT EXISTS `users` (
  `id` int(11) NOT NULL AUTO_INCREMENT,
  `username` varchar(255) NOT NULL,
  `password` varchar(255) NOT NULL,
  `name` varchar(255) NOT NULL,
  `groups` text,
  `settings` text,
  PRIMARY KEY (`id`)
) ENGINE=MyISAM  DEFAULT CHARSET=latin1 AUTO_INCREMENT=2 ;
