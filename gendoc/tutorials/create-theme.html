Tutorial: Create your own theme

Step 1 - Create directories and Manifest

Just copy the default theme and use it as a template: 

cp -r src/themes/default src/themes/mytheme
Then edit metadata.json

Step 2 - Create CSS (LESS)

Just copy the default theme and use it as a template: 

cp src/themes/default.less src/themes/mytheme.less

Step 3 - Add icons and sounds

If you used the command in step 1, you're currently using the default resources via symlinks.

If you want to customize the icons and sounds, just replace the symlinks with normal directories and fill them up.

Step 4 - Build

Just run 

make themes
