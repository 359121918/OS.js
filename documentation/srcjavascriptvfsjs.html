FileDataURL

This is a object you can pass around in VFS when handling DataURL()s (strings). Normally you would use a File, Blob or ArrayBuffer, but this is an alternative. Useful for canvas data etc.

File

This is the Metadata object you have to use when passing files around in the VFS API.
