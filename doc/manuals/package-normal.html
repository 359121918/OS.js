Normal Application

A normal application using the internal OS.js APIs.

Metadata

Key Description Required Default
className The name of your JavaScript namespace Yes  
enabled If your package is enabled or not No true
singular Allow only one instance of the application running No false
autostart If you want to start package on login No false
name The name of your package Yes  
description The description of your package No  
category The category of your package No other
icon The icon filename 
# From icon theme:
"icon": "category/name.png",

# From application directory
"icon": "./name.png",
Yes  
splash Show a splash screen when application starts No true
names Localization of the 'name' attribute 
"names" : {
  "no_NO" : "Norwegian name"
}
No  
descriptions Localization of the 'description' attribute 
"names" : {
  "no_NO" : "Norwegian description"
}
No true
mime What MIME types your package supports (Array with regexp) 
"mime": [
  "^text",
  "inode\\/x\\-empty",
  "application\\/x\\-empty",
  "application\\/x\\-lua",
  "application\\/x\\-python",
  "application\\/javascript",
  "application\\/json"
],
No  
compability Check for these browser features when starting application (Array of strings) 
# See OSjs.Utils.getCompability()

"compabiltiy": [
  "canvas", "blob", "audio", "video"
],
No  
preload The list of resources to preload 
"preload":    [
  {"src": "main.js", "type": "javascript"},
  {"src": "main.css", "type": "stylesheet"},
  {"src": "foo.js", "type": "javascript", combine: false} // Do not combine this script in 'dist'
],
Yes  
depends A list of dependencies you want to load. A dependency is a package name (classname). It only loads the sources, it will not run the application. 
"depends":    [
  "ApplicationSettings"
],
No
