Broadway (GTK3 streaming)

With Broadway you can run native GTK+ 3.x Linux applications in OS.js via WebSockets.

This is not yet ready for production: Github issue

Enable and build

# Enable package
grunt packages:enable:Broadway

# If you run 'dist'
grunt dist

# If you run 'dist-dev'
grunt config manifest # or simply dist-dev

Prepare X server

Run these commands on the server: 

cd Broadway/vendor/
npm install
xhost +
./broadway-server.sh

Usage

To use in OS.js just click the notification icon in panel and connect.
