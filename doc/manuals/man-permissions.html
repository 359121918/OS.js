Setting up Permissions

Note that users with the group "admin" has all prileges!

API methods

You can add groups to API methods.

Simply add them using the configuration system: 

{
  "server": {
    "api": {
      "groups": {
        "mymethodname": "groupname"
      }
    }
  }
}

Then run: 

grunt build:config

User Groups

The "pam", "mysql" and "shadow" handlers all have group permission support.

You can read more about this in the manuals for these handlers.

Blacklist packages for user(s)

Just follow the documentation link about handlers in the "User Groups" section above.
