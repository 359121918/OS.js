Keyboard shortcuts

  • [SHIFT] + TAB
    Switch between open windows
  • [ALT] + H
    Hides current window
  • [ALT] + M
    Maximizes current window
  • [ALT] + R
    Restores current window
  • [ALT] + LEFT
    Move window to the left
  • [ALT] + RIGHT
    Move window to the right
  • [ALT] + UP
    Move window to the top
  • [ALT] + DOWN
    Move window to the bottom
