Configuring and building

You can find all the configuration files in src/conf directory.

The files are loaded in alphabetical order (which is why they are prefixed with a number) and merged together into one final map.

It is highly encouraged that you write your own files instead of modifying the provided ones (because they may be affected by updates).

I provide an example in the repository.

If you make any changes, issue this command to rebuild configuration files (not needed if you run "grunt watch"): 

grunt config

Setting\Getting configuration via Grunt

You can also set (and get) configration via Grunt. 

grunt config:get:client.Connection.RootURI

grunt config:set:client.Connection.RootRUI:/custom/path

grunt config # rebuild config files

Building OS.js

OS.js has two build directories. More information is located in the development environment documentation. To build all changes, simply run: 
grunt
