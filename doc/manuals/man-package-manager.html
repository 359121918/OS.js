Managing system packages/applications

To add a repository, simply clone and add the directory name to a list.

Full list of packages can be found on this Wiki article

Adding repository

# Clone or download into src/packages
git clone --recursive https://github.com/username/package-repository src/packages/package-repository

# Add to repositories list
grunt config:add-repository:package-repository

# Update manifest
grunt manifest

Removing repository

grunt config:remove-repository:package-repository
grunt manifest

Enabling and Disabling Packages

Look at configuration manual page for more information. 

# Enable
grunt packages:enable:PACKAGENAME

# Disable
grunt packages:disable:PACKAGENAME

# Update manifest
grunt manifest

Managing user packages

You can install packages via the Settings application. To remove one simple delete the directory from the Packages folder in your home directory and refresh the metadata.

Currently it supports zip files as a method of installation. Also has a simple, but working app store.

Please note that the Application API is not available here because of security reasons
