Installing OS.js

Please look at INSTALL.md for full instructions. This is just a simplified guide.

There are also automated installers for both UNIX-like systems and Windows. More information in README.md. After running the installer you can skip to "Start" and "View" steps below.

Download

Download and extract or clone with git. Then enter the directory 

# Use a terminal/command-line and find the correct directory where you downloaded
cd OS.js-v2

Install dependencies

Before you can build OS.js you need to install the dependencies.

First you need Grunt: 

# Unix
sudo npm install -g node-cli

# Windows (as administrator)
npm install -g node-cli

Then install dependencies: 

npm install

Build

Simply run: 

grunt

Start

# Unix
./bin/start-dist.sh

# Windows
bin\win-start-dist

View

If the server started successfully simply open http://localhost:8000
