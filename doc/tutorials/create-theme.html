Tutorial: Create your own theme

Create your own Style theme

Simply copy the default theme and modify how you want it: 

mkdir src/themes/styles/mytheme
cp -r src/themes/styles/default/* src/themes/styles/mytheme/
cp -r src/themes/styles/default.less src/themes/styles/mytheme.less

Then update the theme manifest and CSS manually

Create your own Icon theme

Simply copy the default theme and modify how you want it: 

cp -r src/themes/icons/default src/themes/icons/mytheme

Then add your it to build.json

Create your own Sound theme

Simply copy the default theme and modify how you want it: 

cp -r src/themes/sounds/default src/themes/sound/mytheme

Then add your it to build.json

Create your own Font theme

Just add the fonts in to the fonts subfolder and add them to your CSS.

Update your changes

Whenever you add or modify your themes, run this: 
make config themes
