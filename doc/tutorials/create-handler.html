Tutorial: Create your own Handler

The handler is the basis for communication, resource handling, settings and sessions.

There is an Example handler included for you to use as a template. Everything is documented in the Client API reference.

Changing handler

You just have to edit build.json and swap out the configured handler name.

Then run: 

make core config
