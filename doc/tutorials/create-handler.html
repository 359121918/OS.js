Tutorial: Create your own Handler

The handler is the basis for communication, resource handling, settings and sessions.

This tutorial only shows you how to use the included example handler. It's currently only included in PHP server

There is an Example handler included for you to use as a template. It includes a login screen.

Everything is documented in the Client API reference.

Add login HTML template

Copy these files: 

cp src/javascript/handlers/example/index.html dist/example.html
cp src/javascript/handlers/example/index.html dist-dev/example.html

Changing handler

You just have to edit build.json and swap out the configured handler name.

Then run: 

make core config

Setting up database

Create a new mysql user and database, then insert this table: 

CREATE TABLE IF NOT EXISTS `users` (
  `id` int(11) NOT NULL AUTO_INCREMENT,
  `username` varchar(255) NOT NULL,
  `password` varchar(255) NOT NULL,
  `name` varchar(255) NOT NULL,
  `groups` text,
  `settings` text,
  PRIMARY KEY (`id`)
) ENGINE=MyISAM  DEFAULT CHARSET=latin1 AUTO_INCREMENT=2 ;

Apply your connection settings to:

src/server-php/handlers/example/handler.php

IT IS HIGHLY RECOMENDED TO PUT YOUR CONNECTION SETTINGS IN THE ROOT DIRECTORY AND INCLUDE THEM INSTEAD OF HAVING THEM IN THE HANDLER SCRIPT. IT CAN BE READ BY USERS
