Tutorial: Create your own application

Create from template

Simply run:

./obt create-package MyName

This will create some source files for you with usable code. You'll have a basic running application with a Window.

Update manifest

The manifest holds information about your package, like name, description, icon and used resources. Update package.json to fit your application.

This is not required and can be done later

Full example: 

{
  "className":  "ApplicationMyName",               // ClassName used in namespaces
  "enabled":    true,                              // If 'false' application will not be loaded or launchable (Optional)
  "description":"Application Long Description",    // Application description (optional)
  "name":       "Application Name",                // Application name
  "category":   "unknown",                         // Application category (optional)
  "mime":       null,                              // Or an array of regext match MIME strings (Optional)
  "icon":       "apps/file-manager.png",           // Application icon (use ./my-icon.png for icon inside app folder)
  "singular":   false,                             // Allow only one instance ? (Optional)
  "splash":     false,                             // Show a splash screen when loading application (Optional)
  "compability":[],                                // Array of features to check for (Optional, uses Utils.getCompability())
  "preload":    [                                  // A list of files to preload in order to run app
    {"src": "main.js", "type": "javascript"},
    {"src": "main.css", "type": "stylesheet"},
    {"src": "foo.js", "type": "javascript", combine: false} // Do not combine this script in 'dist'
  ]
}

// For translations use
{
  "names" : {
    "no_NO" : "Norwegian name"
  }
  "descriptions" : {
    "no_NO" : "Norwegian description"
  }
}

Install

Now run this command to add it to global manifest (make it visible in OS.js)

make manifest

And, if you're using dist (prodiction environment)

make packages
