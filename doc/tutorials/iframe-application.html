Tutorial: Create IFrame Application

Very useful if you want to convert something you already made or don't want to use the OS.js API. 

grunt create-package:default/MyPackageName:iframe

Example code

In this example we have a simple directory called "data" inside the application with the file "index.html" where your code will be 

function ApplicationExample(args, metadata) {
  Application.apply(this, ['ApplicationExample', args, metadata, {
    src: 'data/index.html',
    title: metadata.name,
    icon: metadata.icon,
    width: 640,
    height: 480,
    allow_resize: false,
    allow_restore: false,
    allow_maximize: false
  }]);
}
