Tutorial: Using VFS

With the VFS you can manipulate files over HTTP. For a full overview of available methods, look at the API documentation.

You can either specify paths as strings or an instance of File Metadata: 

var path = 'home:///file.txt';
// or
var path = new VFS.File('home:///file.txt');

This will show you the basics:

Reading a File

For more information look at the API documentation 

VFS.read(path, function(err, res) {
  if ( err ) {
    alert(err);
    return;
  }

  console.log(res); // The file data
}, {/* options */});

NOTE: If you don't specify the 'type' option this function will return an ArrayBuffer by default

Writing a File

For more information look at the API documentation 

// data can either a 'String', 'File', 'Blob', 'ArrayBuffer', 'JSON' or 'VFS.FileDataURL'
VFS.write(path, data, function(err, res) {
  if ( err || !res ) {
    alert(err || 'Failed to write file');
    return;
  }

  // Success!
}, {/* options */});

Copy a File

For more information look at the API documentation 

VFS.copy(src, dest, function(err, res) {
  if ( err || !res ) {
    alert(err || 'Failed to copy file');
    return;
  }

  // Success!
}, {/* options */});

Scan a directory

For more information look at the API documentation 

VFS.scandir(path, function(err, res) {
  if ( err ) {
    alert(err);
    return;
  }

  console.log(res); // A list of 'VFS.File' instances
}, {/* options */});
