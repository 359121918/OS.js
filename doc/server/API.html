staticAPI.curl(server, args, callback)

cURL API Call 

NOTE: If you do a non-POST request with a body (with type of object) and no
query string was defined in the url, this method will try to transform the
given body data and append to the url.
Parameters:
Name Type Description
server Object Server object
args Object API Call Arguments
Properties
Name Type Attributes Default Description
method String HTTP Call method (GET/POST/HEAD)
url String HTTP Call URL
body Object HTTP POST Payload (alias: query)
timeout Number Timeout in seconds (default=0)
binary Boolean <optional>
 false Return binary (default=false)
mime String <optional>
 If binary, which MIME
headers Object <optional>
 Custom HTTP headers ({key:val})
json Boolean <optional>
 Send request as JSON (autodetected)
contentType String <optional>
 Specify the content-type (autodetected)
callback function Callback function => fn(error, result)
Example
Gives an object like: {httpCode: -1, body: '...'}
