osjs.init

Initializes OS.js APIs and configurations

Parameters

Name Type Description
setup Object Configuration

Options for setup

Name Type Description
port int Listening port (default=null/auto)
dist String Build name (default=dist)
dirname String Server running dir (ex: /osjs/src/server/node)
root String Installation root directory (ex: /osjs)
nw boolean NW build (default=false)
logging boolean Enable logging (default=true)
settings Mixed (Optional) Auto-detected. Path to Settings JSON file or Object
repodir String (Optional) Auto-detected. Package repository root directory (ex: /osjs/src/packages)
distdir String (Optional) Auto-detected. Build root directory (ex: /osjs/dist)

Returns

     Object Returns an object with api, vfs, request, handler and config/setup helpers
